# Image Tiler PWA Structure

```
src/
├── components/          # UI components using Radix
│   ├── ui/             # Base UI components
│   └── features/       # Feature-specific components
├── hooks/              # Custom hooks
├── lib/               # Core utilities & business logic
├── types/             # TypeScript types
└── styles/            # Tailwind & global styles

Key Files:
- components/ui/button.tsx     # Radix-based button component
- components/ui/toggle.tsx     # Toggle switch component
- components/features/dropzone/ # Image dropzone component
- hooks/use-image-processing.ts # Image processing logic
- lib/image-utils.ts          # Image manipulation utilities
```
